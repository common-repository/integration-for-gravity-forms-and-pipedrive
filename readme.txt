Gravity Forms Pipedrive Plugin allows you to quickly integrate Gravity Forms with Pipedrive.

== Description ==

Gravity Forms Pipedrive Add-on sends form submissions from Gravity Forms to Pipedrive CRM when anyone submits a form on your site. You can create a Contact, Organization OR Deal in Pipedrive. Learn more at [crmperks.com](https://www.crmperks.com/plugins/gravity-forms-plugins/gravity-forms-pipedrive-plugin?utm_source=wordpress&utm_medium=gf-pipedrive&utm_campaign=readme)


== Gravity Forms Pipedrive Integration Setup ==

* Go to Forms -> Settings -> Pipedrive then add new account.
* Go to Forms -> select any form -> Settings -> Pipedrive then create new feed.
* Map required Pipedrive fields to Gravity Forms form fields.
* Send your test entry to Pipedrive CRM by clicking "Send to Pipedrive" button.
* Go to Forms -> select any form -> Settings -> Pipedrive Logs and verify status of previously sent entries.


**Connect Pipedrive**

You can connect Pipedrive account to any form by simply entering Pipedrive API key and URL. You can also connect multiple accounts.

**Mapping Contact Fields**

You can easily map Gravity Forms entry fields to any Pipedrive Contact/deal fields. You can also map custom Pipedrive fields.

**Primary Key**

Instead of creating new Object(Contact or Organization or Deal) in Pipedrive, you can update old object by setting Primary Key field.

**Error Reporting**

If there is an error while sending form data to Pipedrive CRM, an email containing the error details will be sent to the specified email address.

**CRM Logs**

Plugin saves a detailed log of each entry whether sent or not sent to Pipedrive. You can easily resend an entry to Pipedrive.

**Export Logs**

You can easily export Pipedrive Logs as a CSV file for  bookkeeping, and accounting purposes.


**Filter Entries**

By default all entries are sent to the Pipedrive, but you can setup rules to limit the entries sent to Pipedrive.

= How to integrate Gravity Forms with Pipedrive =

You can easily integrate Gravity Forms with Pipedrive by following these steps.

* Go to Pipedrive Accounts then connect your account.
* Go to Pipedrive Feeds , select desired Object then map gravity form fields to Pipedrive fields.
* Open any Gravity Forms entry and click "Send to Pipedrive" button , a new member will added in pipedrive.
* Go to Pipedrive logs and check if entry was successfully sent to Pipedrive CRM.
